## 제대로 파는 Git & GitHub - by 얄코

### 프로답게 커밋 관리하기

- 🐰 어떻게 커밋하는게 좋을까요?
- 보다 세심하게 스테이징하고 커밋하기
- 커밋하기 애매한 변화 치워두기
- 커밋 수정하기
- 과거의 커밋들을 수정, 삭제, 병합, 분할하기

### 취소와 되돌리기 보다 깊이 알기

- 관리되지 않는 파일들 삭제하기
- 커밋하지 않은 변경사항 되돌리기
- 🐰 reset했어도 희망은 있다!

### 태그

- 커밋에 🏷️ 태그 달기
- 원격의 🏷️ 태그와 릴리즈

### Branch 보다 깊이 알기

- 🐰 Fastforward vs 3-way merge
- 다른 브랜치에서 원하는 🍒 커밋만 따오기
- 다른 가지의 🌿 잔가지만 가져오기
- 다른 가지의 마디들 🗜️ 묶어서 가져오기
- 협업을 위한 브랜치 활용법
